# DaListener v0.2.0-alpha.3

Save-location visibility update.

## Added

- A save notification after stopping a transcription session.
- The exact local SQLite database path in the notification.
- The exact exported file path after TXT, Markdown, JSON, SRT, or VTT export.
- An **Open folder** button that opens the relevant directory in Windows Explorer.
- A dismiss action and selectable path text.

Alpha.3 also includes alpha.2's window-safe model download log and `pythonw.exe` progress-stream fix.
