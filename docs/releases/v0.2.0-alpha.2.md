# DaListener v0.2.0-alpha.2

Model-preparation visibility and reliability update.

## Fixed

- Replaced the apparently frozen “Preparing model” state with a visible, timestamped live log.
- Added explicit Moonshine file-check, ONNX initialization, Whisper cache/CUDA, and calibration stages.
- Added a two-second heartbeat during long native model initialization.
- Added real downloaded-byte reporting by monitoring Moonshine `.partial` and Hugging Face `.incomplete` cache files.
- Avoided redirecting process stderr, which is unavailable in a windowed executable and caused the alpha.1 logging prototype to fail.

## Install

1. Download `DaListener-0.2.0-alpha.2-windows-x64.zip`.
2. Verify SHA-256: `1c20109672d2737f5105329a6ab7e8dc378850c526b47ff38b42ab5dd0174829`.
3. Extract the entire ZIP.
4. Run `DaListener.exe` and watch the Model Preparation Log during first launch.

The build remains unsigned and does not bundle the approximately 1.3 GB NVIDIA compatibility runtime.
