# DaListener v0.2.0-alpha.1

First public Windows test build.

## Included

- Microphone, Windows system-audio loopback, and simultaneous two-lane capture
- Local Moonshine Small/Medium streaming transcription
- Optional Whisper `large-v3-turbo` GPU finalization
- Automatic hardware inspection and real-speech calibration
- Searchable local session history, bookmarks, and five export formats
- Runtime discovery for pip-installed CUDA 12 cuBLAS and cuDNN 9 libraries

## Install

1. Download `DaListener-0.2.0-alpha.1-windows-x64.zip`.
2. Verify SHA-256: `8b1504756951fdcc31f0fa381e81049469f65dc96f7f05ddbe0a4ef43dd3fc98`.
3. Extract the complete ZIP to a writable directory.
4. Run `DaListener.exe`.
5. Accept the privacy/consent notice and allow the first model download to finish.

The application is unsigned. Windows may display an unknown-publisher warning.

## Validation

- Seven automated tests passed.
- Packaged application completed model loading and calibration during a 20-second smoke test.
- Same-model benchmark: RTX 3090 median `0.288 s`, Ryzen 7 5700X median `6.176 s` for 20 seconds of speech.
- GPU inference was `21.44×` faster in that benchmark and produced the same transcript.

## Known limitations

- The ZIP does not bundle NVIDIA's approximately 1.3 GB CUDA compatibility runtime.
- The app uses Balanced CPU mode unless CUDA 12 cuBLAS and cuDNN 9 are externally available.
- The application and archive are not code-signed.
- Windows audio devices and long sessions still need broader testing.
- Media Foundation reported buffer discontinuities on one tested headphones endpoint.
