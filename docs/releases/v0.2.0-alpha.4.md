# DaListener v0.2.0-alpha.4

This test release adds automatic English and Tagalog transcription and makes plain-text transcripts the visible saved artifact.

## Changes

- Adds **Automatic (English + Tagalog)**, **English**, and **Tagalog** language choices.
- Uses multilingual Whisper to detect English or Tagalog and finalize each completed phrase.
- Keeps Moonshine's low-latency draft lane; Tagalog draft text may be corrected after a pause.
- Uses `large-v3-turbo` on supported NVIDIA CUDA systems, Whisper Small in Balanced CPU mode, and Whisper Tiny in Efficient CPU mode.
- Automatically saves a timestamped `.txt` transcript when listening stops.
- Shows the exact saved file and provides an **Open folder** button.
- Preserves detected-language metadata in internal crash-recovery history.

## Saved transcript location

`%LOCALAPPDATA%\DaListener\Transcripts`

Each filename contains the date and time. Each transcript line contains an elapsed timestamp and source label.

## Notes

This is an unsigned Windows x64 test build. Extract the ZIP before starting `DaListener.exe`. The first use of a transcription mode may download its local model.
